# Task Manager

This is a React project set up with Vite, TypeScript, and Supabase.

## Getting Started

### Prerequisites

Ensure you have the following installed on your local machine:

- Node.js
- npm

### Installation

First, install the dependencies:

```bash
npm install
```

### Start Project

```bash
npm run dev
```

### Build

```bash
npm run build
```

### Test

```bash
npm run test
```

### Lint

```bash
npm run lint
```
